常見問題







Q : 現場候位需要注意甚麼?
A : 現場候位將給予號碼牌，讓顧客留下聯絡資訊及大名，因應疫情入內前幫顧客量測體溫並且做手部清潔消毒。
Q : 有提供素食使用者餐點嗎?
A : 點餐時可以先跟櫃台告知是素食使用者(菜單有素食專區-有提供蛋奶宿及全素)。
Q : 餐廳內有低消嗎?
A : 每桌至少消費一份鍋物，共鍋者額外收取該鍋物一半的清潔使用費。
Q : 營業時間是幾點開始?
A : 營業時間為中午時段11:00-15:00(最晚點餐時間為14:00)            晚餐時段17:00-21:00(最晚點餐時間為20:00)
Q : 請問可以用刷卡或手機支付嗎?
A : 本店提有提供現金付款、信用卡、街口支付、apple pay、samsung pay等多元支 付工具。
Q : 食材是從哪個國家進口的呢?
A : 本店使用牛肉 : 澳洲和牛、日本和牛 本店使用豬肉 : 西班牙伊比利豬、台灣豬肉(不含萊客多巴胺) 本店使用雞肉 : 台灣土雞肉
